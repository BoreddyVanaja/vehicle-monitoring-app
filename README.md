# 🚗 Vehicle Load Monitoring System

## 📌 Overview
This project is an IoT-based Vehicle Load Monitoring System designed to track and monitor vehicle load in real-time. It helps prevent overloading and improves road safety by sending live data to a web dashboard.

 
- 📡 Real-time vehicle load monitoring
- 🌐 Live dashboard using web interface
- 🔥 Firebase integration for data storage
- ⚡ MQTT protocol for fast data transmission
- 📊 Dynamic updates without refreshing the page

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Firebase (Firestore / Realtime DB)
- MQTT Protocol
- IoT Simulation (ESP32 / Wokwi)
